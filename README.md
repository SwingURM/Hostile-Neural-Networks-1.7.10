# Hostile Neural Networks (Forge 1.7.10 移植版)

将 [Hostile Neural Networks](https://github.com/Shadows-of-Fire/Hostile-Neural-Networks) 从 NeoForge 1.21.1 移植到 Minecraft Forge 1.7.10 (GTNH)。

基于 [ExampleMod1.7.10](https://github.com/GTNewHorizons/ExampleMod1.7.10) 模板。
